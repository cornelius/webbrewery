The Data File

The data file defines data resources, i.e. data files not subject to processing by the WebBrewery. Each line of the data file defines a data ressource using the following format:

OPERATION: location

There are two kinds of operations: COPY: means that the resource (a file or directory) at the given location is copied from the src directory to the output directory. LINK: means that the resource (a file or directory) is linked from the output directory to the given location unless it already exists as real file or directory at the output directory.
