The Figures File

The figure file contains definitions of symbols for figure resources, i.e. images. Each line defines one figure resource. Its format is

figuresymbol figurelocation

The figuresymbol defines a symbol, which can be used in the source file with the FIG: symbol. The figurelocation defines the address of the figure ressource. It is interpreted local unless a URL starting with http:// is given. I recommend using a hierachical naming scheme for the symbols as described for the link symbols (see Link File).

Lines starting with # are treated as comments and define no link symbol. Empty lines are ignored.
