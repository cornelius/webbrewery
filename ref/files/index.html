Configuration Files

The following configuration, index or database files are required for processing of source files by the WebBrewery. They contain basic information common to possibly many pages of the site under development.

  • The Content file describes the structure of the web site.
  • The Links file contains the actual URLs of all hyperlinks used in the web site.
  • The Figures file contains the actual URLs of all referenced image elements of the web site.
  • The Data file contains references for non-HTML data associated with the web site.
